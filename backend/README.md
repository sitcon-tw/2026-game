# 如何啟動

## 如果你是寫前端的

你要看 OpenAPI docs 請打：

```bash
make run-docs
```

注意一下這邊什麼都沒有喔，他就只是個 html 你並沒有真的把後端跑起來 :D

## 先啟動 database:

```bash
docker compose -f compose.dev.yml up -d
```

## 再啟動 backend:

```bash
make run
```

如果你看到類似：
```
2026-02-04T12:25:27.940+0800    INFO    cmd/main.go:48  Starting server {"port": "8000", "env": "dev"}
```
那就代表你成功啟動這個 backend server 了

# 大地遊戲企劃書

## 遊玩方式
點擊畫面上的「播放」按鈕，遊戲畫面上其中一個按鈕會亮起，並隨節奏撥放樂曲。
遊戲會依序點亮顏色並播放獨特的樂曲，會眾需觀察按鈕之顏色亮起的順序，當序列播放完畢後，會眾需依播放順序點擊按鈕。若順序完全相同，則成功通關。若有任一不同則通關失敗，可重新開始關卡。
起始已解鎖關卡為 5 關，會眾需透過與攤位互動、打卡、認識新朋友等方式解鎖新關卡（詳見「關卡獲得」部分）
第一關按鈕為 2 個，2~5 關 4 個，後續每五關增加四個按鈕，最多為 32 個按鈕。

## 排行榜
- 顯示用戶名稱及該用戶已完成的關卡數。
- 以完成關卡的數量排名。若遇相同，則以完成的先後順序排名。
- 顯示前10名、您當前的排名、您當前的排名+-5。
- 截止時間：紀念品攤收掉前 30 分鐘。（約 16.:00）

## 關卡獲得
皆僅第一次可解鎖關卡，後續重複與同一攤位、場地、朋友互動不會再獲得新關卡。
解鎖關卡並不等於完成關卡，僅僅是獲得此關的挑戰資格。
![IMG_1528 2](https://hackmd.io/_uploads/SkvrFOYQbx.jpg =30%x)
> 上方無播放鍵和問號
1. 跟攤位互動：與社群攤位交流互動，提供 Qrcode 給對方掃描。獲得 X 關。
點擊此欄位後顯示所有攤位，未互動過的攤位較暗、已互動過的攤位較亮。
![IMG_1519 3](https://hackmd.io/_uploads/BkYUFdYmZg.jpg =30%x)
> （上方無播放鍵和問號）
2. 闖關：參加咖啡廳、導遊團、指南針計畫。
4. 打卡：到其他活動所在場地打卡並掃描 Qrcode。獲得 X 關。
點擊此欄位後顯示所有活動和所在場地，未打卡過的活動較暗、已打卡過的活動較亮。
提供打卡的活動包括：Demo 展、白板大戰
3. 認識新朋友：與其他會眾互動，提供 Qrcode 給對方掃描或掃描對方的Qrcode。認識一位解鎖一關。
透過此方法解鎖關卡具數量限制，與一攤位互動後可再以認識20個新朋友解鎖關卡。若超過此數量，後續認識的新朋友將不會解鎖新關卡。
與所有攤位互動過一次、打卡過一次所有場地後則解除限制。
未超過數量時，點擊此欄位後跳轉至 Qrcode 介面。
超過數量後，畫面將會顯示「你太E了」、「請透過其他方式獲得 X 關後才可再透過認識朋友解鎖關卡」。下方會有「仍要認識」按鈕，若點擊，也會跳轉至 Qrcode 介面（但掃描後不會解鎖關卡）。
![IMG_1521](https://hackmd.io/_uploads/S1wvYOFmZl.jpg =30%x)
> （上方無播放鍵和問號）

> 註解：所有的 X都是一樣的，待確認年會攤位數量後確定具體數據。

## 獎勵領取
**介面設計**
- 折價卷獲得進程
- 已擁有折價卷
- 紀念品攤收掉前 30 分鐘（約 16:00）要呈現無法再獲得折價卷的狀態，因此於介面需有設計說明
    - 說明文案：折價卷獲得期限僅至 16:00！把握機會與更多攤位互動、完成更多關卡，已獲得更多折扣！

**兌換方式**
- 掃描會眾 Qrcode（Qrcode 介面的那個）
- 折價卷只能一次全部使用完

**折價多少**
具體折抵金額需再跟設計組溝通，ABCD 呈遞減。數據等攤位數量確認後可能會微調
- 折抵 A 元：排行榜前三名
- 折抵 B 元：打卡完所有攤位和活動場地
- 折抵 C 元：排行榜前 30% （預計約 150 份）
- 折抵 D 元：排行榜前 60% （預計約 300 份）
- 折抵 E 元：完成 10 關（預計約 500 份）
