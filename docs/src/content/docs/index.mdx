## 角色分類

- `會眾`
- `攤位`
- `移動攤位`
- `紀念品收銀員`
- `管理員`
- `打卡點`

## 文件範圍

- 內容以目前前端程式碼可見功能為主，不延伸推測尚未實作的後台或現場 SOP。
- 若同一功能存在不同頁面，文件會以「實際操作入口」為優先整理，並在不同頁面互相連結。
- 若某個角色目前沒有獨立頁面，會在該角色頁中直接註明限制與替代流程。
