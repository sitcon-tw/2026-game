## 對應頁面

- `/scan`
- `/play/friends`

## `/scan` 的核心定位

這個頁面同時扮演兩個角色：一個是「我的名片頁」，另一個是「掃描器入口」。前端採用名片優先的設計，玩家一進來先看到自己的公開資訊與 QR Code。

## 我的名片功能

- 顯示暱稱、頭像、目前關卡
- 顯示自介、公開信箱、外部連結
- 顯示一次性 QR Code
- 可放大 QR Code 給別人掃
- 可打開編輯視窗更新名片資訊

## 掃描器功能

- 玩家可以從名片頁切到掃描器模式
- 掃描 `qru1.` 開頭的 QR Code 時，前端會走加好友流程
- 掃描其他活動 QR Code 時，前端會走活動打卡流程
- 成功與失敗都會顯示即時提示訊息

## 好友額度資訊

- 畫面會顯示目前好友數量與上限
- 也會換算還剩幾位朋友可以新增
- 如果已到上限，前端會提示「打卡更多攤位，解鎖加朋友額度」

## `/play/friends` 的功能

- 顯示好友列表
- 一頁 5 位好友
- 可查看頁碼並切換上一頁、下一頁
- 點擊好友可打開對方公開名牌
- 頁面底部可直接跳去 `/scan` 掃描好友 QR Code
- 也可從這裡再打開更新自己名牌的視窗

## 文件上的解讀

- `/scan` 是操作入口
- `/play/friends` 是整理與回顧入口
- 兩頁配合後，才構成完整的社交互動流程
