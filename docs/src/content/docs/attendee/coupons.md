title: 折價券

---

## 對應頁面

- `/coupon`

## 功能目的

這個頁面整合了玩家所有與折價券有關的事情，包括查看票券、了解哪些尚未解鎖、產生核銷 QR Code，以及輸入 gift coupon 代碼。

## 頁面上會看到什麼

- 未使用折價券
- 已使用折價券
- 尚未解鎖的折價券定義
- 未使用張數
- 未使用總折抵金額
- 使用規則

## 票券列表的呈現方式

- 如果玩家已經擁有某種折價券，頁面會依實際持有狀態顯示為未使用或已使用
- 如果玩家尚未獲得某張票券，但系統有定義這種獎勵，頁面會顯示為 locked
- 如果玩家拿到的是 gift coupon 這類不在定義表中的票券，前端也會額外加進列表

## 出示給紀念品攤位掃描

- 當玩家有未使用折價券，而且自己的 `coupon_token` 存在時，頁面會出現 `兌換折價券` 按鈕
- 點擊後會打開專用 QR Code modal
- modal 會顯示總折抵金額與給工作人員掃描的 QR Code
- 頁面也會提醒玩家要到 2F 紀念品攤位使用

## 手動輸入兌換代碼

- 玩家可以展開輸入框，手動輸入 gift coupon code
- 成功後折價券會加入列表
- 失敗時會顯示錯誤訊息

## 規則區塊目前寫死顯示的限制

- 限時取得期限
- 使用截止時間
- 最低消費門檻
- 單次核銷時會折抵當下所有折價券
