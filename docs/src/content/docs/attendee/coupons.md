---
title: 折價券
description: 玩家如何查看、出示與兌換自己的折價券。
---

折價券頁面顯示你目前擁有的所有折價券。頁面頂部會告訴你有幾張可用、下方以堆疊卡片方式依序列出未使用、已使用和尚未解鎖的票券。

<img src="/docs/attendee/coupon.png" alt="折價券頁面" width="50%" />

如果你有未使用的折價券，頁面下方會出現「兌換折價券」按鈕，點擊後會顯示 QR Code 給紀念品攤位工作人員掃描。

## 對應頁面

- `/coupon`

## 頁面頂部資訊

「你有 **X** 張可用折價券」（可用張數以粗體大字顯示）

## 兌換規則區塊

頁面上固定顯示六項規則：

| 圖示 | 規則 |
|------|------|
| ⚡ | 限時動態或貼文分享 10:00–12:00 |
| 🏆 | 排行榜結算 16:00 |
| 🕐 | 獲得期限 至 16:00 |
| 🛍️ | 使用期限 至 16:30 收攤 |
| 🎫 | 門檻 單筆滿 200 元折抵 |
| ℹ️ | 單次使用 折抵所有折價券 |

## 票券列表的三種狀態

| 狀態 | 說明 |
|------|------|
| 未使用 | 顯示折抵金額與票券說明，可用於兌換 |
| 已使用 | 灰色顯示，標示已使用日期 |
| 尚未解鎖（locked）| 顯示所需通過關卡數，淡色鎖定圖示 |

> 如果你拿到的是 gift coupon（例如從活動取得），即使不在定義清單中，也會額外加進列表。

## 出示給紀念品攤位

1. 確認你有未使用折價券
2. 點擊「兌換折價券」按鈕（位於頁面下方）
3. 彈窗顯示：
   - 大字金色總折抵金額（例如「150 元」）
   - 標題：「折價券兌換明細」
   - 說明：「可折抵總額共 X 元」
   - QR Code：「出示 QR Code 供工作人員掃描」
   - 地點提醒：「請至 2F 紀念品攤位使用」
4. 出示 QR Code 請工作人員掃描，完成核銷

## 手動輸入兌換代碼

如果你有 gift coupon 代碼：

1. 點擊「有兌換代碼？」展開輸入框
2. 在「輸入兌換代碼」欄位貼上代碼
3. 點擊「兌換」按鈕
4. 成功：折價券加入列表
5. 失敗：顯示對應的錯誤訊息

## 後端行為

- 取得持有折價券：`GET /api/discount-coupons/`
- 取得折價券定義：`GET /api/discount-coupons/coupons`（列出所有獎勵定義）
- 兌換 gift coupon：`POST /api/discount-coupons/gifts`（body: `{code}`）
- 核銷 QR Code：由工作人員端 `POST /api/discount-coupons/staff/redemptions` 觸發
