## 使用方式

- 如果要了解玩家一開始怎麼進入遊戲，先看「登入與進入遊戲」。
- 如果要了解關卡怎麼玩，先看「關卡列表與闖關規則」與「單一關卡挑戰流程」。
- 如果要了解怎麼解鎖更多關卡，從「解鎖關卡總覽」往下讀各種來源。
- 如果要了解現場互動與社交功能，重點在「好友、名片與掃描」與「指南針計畫」。
- 如果要了解兌換福利，直接看「折價券」。
