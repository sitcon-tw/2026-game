title: 闖關活動解鎖
description: 玩家如何查看 challenge 類型活動的完成狀態。
---

## 對應頁面

- `/play/challenges`

## 功能目的

這個頁面專門顯示 `challenge` 類型活動，不是主線音樂關卡本身，而是被當作解鎖來源的活動清單。

## 頁面內容

- 使用格狀按鈕列出所有 challenge 類活動
- 已完成與未完成會用不同顏色與透明度呈現
- 點擊單一項目會跳出 modal

## 詳細視窗內容

- 活動名稱
- 是否完成

## 文件重點

- 這頁的目的在於查狀態，不是直接開始活動
- 玩家如果要知道還差哪些 challenge 類解鎖來源，應該看這頁
