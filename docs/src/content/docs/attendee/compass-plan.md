指南針計畫是一個小組互動功能。如果你有加入指南針計畫分組，這個頁面讓你和同組夥伴互相掃描 QR Code 完成互簽，也可以查看全組成員的遊戲進度。

可以在解鎖關卡方式找到入口

如果你還沒有分組，頁面只會顯示：「你目前沒有參與指南針計畫。看到這個畫面代表你尚未分組。」

## 對應頁面

- `/play/compass`

## 功能前提

只有當你的帳號有 `group` 欄位時，才會看到完整的指南針計畫功能。沒有分組的玩家只會看到上述提示。

## 頁面頂部資訊

有分組的玩家會看到：

- 頁面標題：「指南針計畫」
- 小組名稱：「你的指南針小組：[小組名稱]」

## 兩個分頁

頁面分為兩個分頁（以兩欄按鈕切換）：

| 分頁     | 功能                                  |
| -------- | ------------------------------------- |
| 掃描簽到 | 掃描夥伴 QR Code 或出示自己的 QR Code |
| 夥伴列表 | 查看全組成員狀態                      |

## 掃描簽到分頁

- 相機啟動，掃描夥伴的「我的 QR Code」完成互簽
- 說明文字：「掃描指南針夥伴的「我的 QR Code」完成簽到 / 每位夥伴只能互簽一次」
- 進度顯示：「目前進度 X/Y」（已簽到人數 / 全組人數）
- 切換按鈕：
  - 「顯示我的 QR Code」→ 顯示你的 QR Code 讓夥伴掃
  - 「切回掃描簽到」→ 回到掃描器模式

## 夥伴列表分頁

- 列出所有組員：頭像、暱稱、目前關卡（第 X 關）
- 每位成員顯示簽到狀態：
  - 已簽到（綠色徽章）
  - 未簽到（灰色徽章）
- 點擊其他成員可打開其公開名牌
- 若尚無其他組員：「目前還沒有其他指南針夥伴。」

## 這個功能的角色

- 互簽進度計入 `/play` 解鎖關卡總覽的「指南針計畫」卡片
- 掃描方式與 `/scan` 的 QR Code 相同（`qru1.` 開頭的 QR）
- 觸發 API：`POST /api/group/check-ins`（雙向互簽，每對只能互簽一次）
