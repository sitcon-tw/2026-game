## 對應頁面

- `/play`

## 功能目的

這個頁面是「如何解鎖更多關卡」的入口總表。前端會把所有可計算的來源彙整成卡片，玩家先看這裡，再進入個別細項頁。

## 目前會出現的來源

- 攤位
- 闖關
- 打卡
- 認識新朋友
- 指南針計畫（只有有分組的玩家看得到）

## 卡片上顯示什麼

- 目前完成數
- 總數
- 是否已載入資料
- 點擊後要前往的詳細頁

## 點進去後對應頁面

- 攤位 -> `/play/booths`
- 闖關 -> `/play/challenges`
- 打卡 -> `/play/checkins`
- 認識新朋友 -> `/play/friends`
- 指南針計畫 -> `/play/compass`

## 為什麼這頁重要

玩家在關卡卡住時，不一定知道還有哪些方式可以繼續解鎖。這個頁面就是把所有進度來源放在一起，讓玩家快速判斷下一步要去哪一區互動。
