---

## 對應頁面

- `/admin/assign`

## 功能目的

這個頁面提供人工補單能力。管理員可以直接指定某位玩家，手動發一張折價券給他。

## 操作流程

1. 用暱稱搜尋玩家。
2. 從搜尋結果中選定一位玩家。
3. 輸入 `discount_id`。
4. 輸入金額。
5. 送出指派。

## 頁面上顯示的玩家資訊

- 暱稱
- 當前關卡
- 部分 user id

## 文件重點

- 這頁不是透過掃碼完成，而是管理員直接操作
- 很適合處理例外狀況、補發或人工調整
